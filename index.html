Dreadfort

The Multi-Tenanted Python Event Logging Service

About Dreadfort

Dreadfort is a multi-tenanted Python based system for event collection, transit and processing in the large. It's primary use case is for large-scale logging but can be used in many other scenarios including usage reporting and API tracing.

Dreadfort is an active fork of the original project known as Meniscus
